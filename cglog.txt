# ⚠️ Issues & Fixes

## ❌ Frontend Issues

### 1. Blank Screen in React
- Cause: Rendering mismatch / incorrect component setup
- Fix: Proper component structure + routing fixes

### 2. Icon Mismatch
- Cause: Incorrect icon imports / library mismatch
- Fix: Standardized icon usage

---

## ❌ Backend Issues

### 3. Chat Notifications Not Broadcasting
- Cause: Messages were not reaching all users
- Fix:
  - Implemented WebSocket personal topics:
    ```
    /user/{userId}/...
    ```

---

### 4. Data Leak Across Projects
- Cause: Improper repository queries
- Fix:
  - Strict filtering using userId + projectId
  - Enforced data isolation at query level

---

### 5. Legacy Alerts
- Issue: Old alert() based system
- Fix:
  - Replaced with modern Toast Notifications
